# Security Policy

## Supported Versions

This project follows a rolling release model. Only the latest commit on the default branch is supported for security fixes.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly and do not open a public issue.

- Contact: [reuben@hummbl.io](mailto:reuben@hummbl.io)
- Please include: a detailed description, impact, reproduction steps, and any supporting logs or screenshots.

We will acknowledge receipt within 3 business days and provide a remediation plan or timeline as soon as possible.

## Disclosure

We follow coordinated disclosure. Please allow a reasonable window for triage and fixes before public disclosure.
