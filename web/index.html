Add to Claude Desktop (30 seconds)

Install the MCP server to give Claude access to all 120 models:

1. Install

npm install -g @hummbl/mcp-server

2. Add to config

Edit claude_desktop_config.json:

{
  "mcpServers": {
    "hummbl": {
      "command": "npx",
      "args": ["@hummbl/mcp-server"]
    }
  }
}

3. Restart Claude

The tools appear in Claude's attachment menu.

  • hummbl_search - Find models by problem
  • hummbl_get_model - Get model details
  • hummbl_recommend - Get recommendations

What is Base120?

A validated framework of 120 mental models organized across 6 transformations:

  • P (Perspective) - Frame and anchor viewpoint
  • IN (Inversion) - Flip to see what avoiding failure reveals
  • CO (Composition) - Combine for emergent properties
  • DE (Decomposition) - Break complexity down
  • RE (Recursion) - Apply patterns at multiple scales
  • SY (Systems) - Understand governing rules

Who is this for?

You're building AI agents that need to think better. Your agents use LangChain, CrewAI, or custom frameworks. You want structured reasoning, not just LLM autocomplete.
